# API

### GET

1. 순위검색 ( /ranking )

    종합기록, 다승, 국수, 승률, 승점률, 1,2위률, 4위률 확인

2. 기록 ( /record_list )

    등록한 기록들 확인
    
3. 개인성적 ( /record_per )

    동장기록, 남장기록, 월간, 주간, 요일별, 시간별 성적확인
       
4. 상대성적 ( /record_versus_res )

    동장기록, 남장기록 성적확인

5. 사용자목록 ( /player )

    동장기록, 남장기록, 월간, 주간, 요일별, 시간별 성적확인
